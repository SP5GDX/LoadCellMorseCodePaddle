# LoadCellMorseCodePaddle
Load cell morse code paddle

This is a concept of using load cell as Morse Code paddle
Advantages :
- easy mechanical apply
- no moving parts
- fast and qiet
- cheap - Trinket + laod cells + AD ~25$
- auto calibration - so readjustment needed
